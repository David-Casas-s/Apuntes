# LIMIT:

La clusula LIMIT se usa para especificar el numero maximo de filas que se mostrara en el resultado de una consulta SQL, su sintaxis basica es:

```SQL
SELECT "columna1" FROM "nameTab" LIMIT numero_filas;
```
